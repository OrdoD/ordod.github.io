Tripartite Project FA22

Welcome to the About page for the Tripartite Project for Fall 2022!

About

The Tripartite Project FA22 is a project developed by students of the Real World Project Studio (RWPS) class at the University of Silicon Valley (USV). The main theme of the project is Digital Twinning. Which consists of a virtual model designed to accurately reflect a physical object. In the case of this project, students of the FA22-23 RWPS course will be creating a digital twin of a person. The digital twin of the person will be implemented into a rule-based chatbot.
